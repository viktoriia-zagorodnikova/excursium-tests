# Автотесты для сайта Excursium

## Описание
Проект автоматизированного тестирования фронт-сайта магазина экскурсий Excursium.

## Стек технологий
- Python
- Pytest
- Selenium WebDriver

## Структура проекта
- `conftest.py` — настройка драйвера
- `test_auth.py` — тесты авторизации (EX-001, EX-002)
- `test_register.py` — тесты регистрации (EX-009, EX-013)
- `test_filter.py` — тесты фильтрации (EX-014, EX-017)

## Что тестируется
- Авторизация по email и паролю
- Регистрация нового пользователя
